# 仿淘宝,京东红包雨(基于Phaser框架)


----------
本红包雨项目是基于HTML5的游戏框架Phaser写的,最终形成的是一个canvas,所以性能很好,但是必须要说的是这个框架比较大,压缩后也有700K左右,所以请慎用.

phaser游戏开发脚手架，支持ES6语法，支持class类继承

**develope(开发):**
```
1. npm install
2. npm run dev
3. open localhost:8081 in chrome
```
